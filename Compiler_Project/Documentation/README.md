# مترجم اللغة البرمجة العربية - Arabic Programming Language Compiler

## نظرة عامة

مشروع مترجم كامل للغة برمجة عربية، مصمم ليدعم كتابة البرامج باللغة العربية مع الترجمة إلى كود MIPS. المشروع مبني باستخدام C للجزء الأساسي من المترجم و C# لواجهة المستخدم.

## الميزات

- **لغة برمجة عربية**: دعم كامل للكتابة باللغة العربية
- **مترجم متكامل**: محلل معجمي، نحوي، دلالي، ومولد كود
- **ترجمة إلى MIPS**: إنتاج كود MIPS قابل للتنفيذ
- **واجهة مستخدم حديثة**: واجهة رسومية سهلة الاستخدام
- **دعم Unicode**: دعم كامل للغة العربية
- **عرض مرئي**: عرض شجرة الإعراب وشجرة البنية المجردة
- **إدارة الأخطاء**: نظام شامل لتتبع وإبلاغ الأخطاء

## التثبيت

### 1. استنساخ المشروع
```bash
git clone https://github.com/yourusername/arabic-compiler.git
cd arabic-compiler
```

### 2. بناء الجزء الأساسي (C)
```bash
cd CompilerCore
make
```

### 3. بناء واجهة المستخدم (C#)
```bash
cd ../GUI
dotnet build
```

### 4. تشغيل التطبيق
```bash
dotnet run
```

## استخدام المشروع

### 1. كتابة برنامج باللغة العربية
```arabic
برنامج
[
    صحيح العداد = 0 ؛
]:
    طالما العداد < 10
    [
        اطبع العداد ؛
        العداد = العداد + 1 ؛
    ] ؛
    اطبع "انتهى العد" ؛
؛
```

### 2. ترجمة البرنامج
- افتح التطبيق
- اكتب الكود في محرر الأكواد
- اضغط على زر "ترجمة"
- شاهد النتائج في عارض كود MIPS

### 3. عرض شجرة الإعراب
- اضغط على "عرض شجرة الإعراب"
- شاهد البنية النحوية للبرنامج

## مواصفات اللغة

### الكلمات المحجوزة
- `برنامج` - بداية البرنامج
- `إذا` - عبارة شرطية
- `طالما` - حلقة تكرار
- `اطبع` - إخراج
- `اقرأ` - إدخال

### أنواع البيانات
- `صحيح` - أعداد صحيحة
- `حقيقي` - أعداد حقيقية
- `حرف` - أحرف مفردة
- `منطقي` - قيم منطقية
- `رمزي` - نصوص

### العوامل
- `+`, `-`, `*`, `/` - عمليات حسابية
- `==`, `!=`, `<`, `>` - مقارنات
- `&&`, `||`, `!` - عمليات منطقية

## البنية المعمارية

### تدفق البيانات
```
الكود المصدري → المحلل المعجمي → المحلل النحوي → المحلل الدلالي → مولد الكود → كود MIPS
```

### المكونات
1. **المحلل المعجمي**: تحويل النص إلى رموز
2. **المحلل النحوي**: بناء شجرة الإعراب
3. **المحلل الدلالي**: فحص صحة البرنامج
4. **مولد الكود**: إنتاج كود MIPS
