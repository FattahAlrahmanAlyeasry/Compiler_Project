# دليل المستخدم لمترجم اللغة البرمجة العربية
## User Guide for the Arabic Programming Language Compiler

### نظرة عامة (Overview)
هذا الدليل يوضح كيفية استخدام مترجم اللغة البرمجة العربية. المترجم يوفر واجهة مستخدم رسومية سهلة الاستخدام تسمح بكتابة وتحرير وترجمة البرامج العربية إلى كود MIPS.

This guide explains how to use the Arabic Programming Language Compiler. The compiler provides a user-friendly graphical interface that allows writing, editing, and compiling Arabic programs into MIPS code.

### متطلبات النظام (System Requirements)

#### متطلبات الأجهزة (Hardware Requirements):
- **المعالج**: Intel Pentium 4 أو أحدث
- **الذاكرة**: 2 GB RAM على الأقل
- **مساحة القرص**: 100 MB متاحة
- **الشاشة**: دقة 1024×768 أو أعلى

#### متطلبات البرامج (Software Requirements):
- **نظام التشغيل**: Windows 7/8/10/11
- **Visual Studio**: Visual Studio 2015 أو أحدث
- **.NET Framework**: .NET Framework 4.5.2 أو أحدث
- **مترجم C++**: Visual Studio C++ Compiler

### تثبيت المترجم (Installing the Compiler)

#### 1. تحميل الملفات
```
1. قم بتحميل ملف arabic_compiler.dll
2. قم بتحميل ملف ArabicCompiler.exe
3. تأكد من وجود ملفات .NET Framework المطلوبة
```

#### 2. تثبيت الملفات
```
1. انسخ arabic_compiler.dll إلى مجلد النظام (System32)
2. انسخ ArabicCompiler.exe إلى مجلد العمل
3. تأكد من أن جميع الملفات في نفس المجلد
```

#### 3. التحقق من التثبيت
```
1. شغل ArabicCompiler.exe
2. تأكد من عدم ظهور رسائل خطأ
3. تحقق من عمل جميع القوائم
```

### كتابة البرامج العربية (Writing Arabic Programs)

#### 1. هيكل البرنامج الأساسي (Basic Program Structure)

##### نموذج برنامج بسيط:
```
برنامج اسم_البرنامج؛
[
  // تعريفات (اختيارية)
  متغيرات:
    متغير1: نوع1؛
    متغير2: نوع2؛
]:
  // العبارات البرمجية
  اطبع("مرحباً بالعالم!")؛
]
نهاية؛
```

#### 2. أنواع البيانات (Data Types)

##### الأنواع الأساسية:
```
عدد: أعداد صحيحة (مثل: 42، -17، 0)
حقيقي: أعداد عشرية (مثل: 3.14، -2.5، 0.0)
نص: سلاسل نصية (مثل: "مرحباً"، "عالم")
منطقي: قيم منطقية (صحيح، خطأ)
حرف: حرف واحد (مثل: 'أ'، 'ب')
```

##### أمثلة على التعريفات:
```
متغيرات:
  اسم: نص؛
  عمر: عدد؛
  طول: حقيقي؛
  نشط: منطقي؛
```

#### 3. العبارات البرمجية (Programming Statements)

##### عبارة الإسناد (Assignment):
```
متغير := قيمة؛
مثال: عمر := 25؛
```

##### عبارة الإخراج (Output):
```
اطبع(تعبير1، تعبير2، ...)؛
مثال: اطبع("العمر: "، عمر)؛
```

##### عبارة الإدخال (Input):
```
اقرأ(متغير1، متغير2، ...)؛
مثال: اقرأ(اسم، عمر)؛
```

##### عبارة الشرط (Conditional):
```
إذا شرط فإن
  عبارة1؛
وإلا
  عبارة2؛
نهاية؛
```

##### عبارة الحلقة (Loop):
```
طالما شرط كرر
  عبارة؛
نهاية؛
```

#### 4. أمثلة على البرامج (Program Examples)

##### مثال 1: برنامج الترحيب
```
برنامج مرحبا؛
[
  متغيرات:
    اسم: نص؛
    عمر: عدد؛
]:
  اطبع("أدخل اسمك: ")؛
  اقرأ(اسم)؛
  اطبع("أدخل عمرك: ")؛
  اقرأ(عمر)؛
  اطبع("مرحباً "، اسم، "، عمرك "، عمر، " سنة")؛
]
نهاية؛
```

##### مثال 2: برنامج حسابي
```
برنامج حسابي؛
[
  متغيرات:
    أ: عدد؛
    ب: عدد؛
    مجموع: عدد؛
]:
  اطبع("أدخل الرقم الأول: ")؛
  اقرأ(أ)؛
  اطبع("أدخل الرقم الثاني: ")؛
  اقرأ(ب)؛
  مجموع := أ + ب؛
  اطبع("المجموع: "، مجموع)؛
]
نهاية؛
```

##### مثال 3: برنامج مع شروط
```
برنامج تقييم؛
[
  متغيرات:
    درجة: عدد؛
]:
  اطبع("أدخل الدرجة: ")؛
  اقرأ(درجة)؛
  إذا درجة >= 90 فإن
    اطبع("ممتاز")؛
  وإلا إذا درجة >= 80 فإن
    اطبع("جيد جداً")؛
  وإلا إذا درجة >= 70 فإن
    اطبع("جيد")؛
  وإلا
    اطبع("مقبول")؛
  نهاية؛
]
نهاية؛
```

##### مثال 4: برنامج مع حلقات
```
برنامج جدول_الضرب؛
[
  متغيرات:
    رقم: عدد؛
    عداد: عدد؛
]:
  اطبع("أدخل رقماً: ")؛
  اقرأ(رقم)؛
  عداد := 1؛
  طالما عداد <= 10 كرر
    اطبع(رقم، " × "، عداد، " = "، رقم * عداد)؛
    عداد := عداد + 1؛
  نهاية؛
]
نهاية؛
```

### عملية الترجمة (Compilation Process)

#### 1. بدء الترجمة (Starting Compilation)

##### الطرق المتاحة:
1. **قائمة الترجمة**: ترجمة → ترجمة
2. **شريط الأدوات**: زر الترجمة
3. **اختصار لوحة المفاتيح**: Ctrl+F5

#### 2. مراحل الترجمة (Compilation Stages)

##### المرحلة 1: التحليل المعجمي
```
- قراءة الكود المصدري
- تقسيمه إلى رموز
- التعرف على الكلمات المحجوزة
- إنشاء قائمة الرموز
```

##### المرحلة 2: التحليل النحوي
```
- بناء شجرة الإعراب
- التحقق من صحة البنية
- الكشف عن الأخطاء النحوية
```

##### المرحلة 3: التحليل الدلالي
```
- بناء جدول الرموز
- فحص أنواع البيانات
- التحقق من صحة العمليات
```

##### المرحلة 4: توليد الكود
```
- بناء شجرة البنية المجردة
- توليد كود MIPS
- تحسين الكود الناتج
```

#### 3. مراقبة عملية الترجمة (Monitoring Compilation)

##### مؤشرات التقدم:
- **شريط التقدم**: يوضح المرحلة الحالية
- **رسائل الحالة**: معلومات عن كل مرحلة
- **عداد الأخطاء**: عدد الأخطاء المكتشفة

##### رسائل النجاح:
```
تمت الترجمة بنجاح
عدد الأخطاء: 0
تم إنشاء كود MIPS
```

### عرض النتائج (Viewing Results)

#### 1. عرض كود MIPS (Viewing MIPS Code)

##### نافذة كود MIPS:
```
┌─────────────────────────────────────────────────────────────┐
│                        كود MIPS                            │
├─────────────────────────────────────────────────────────────┤
│ .text                                                      │
│ .globl main                                               │
│ main:                                                     │
│   # تهيئة المكدس                                          │
│   addi $sp, $sp, -8                                       │
│   sw $ra, 4($sp)                                          │
│   sw $fp, 0($sp)                                          │
│   move $fp, $sp                                            │
│                                                            │
│   # طباعة الرسالة                                          │
│   la $a0, msg1                                            │
│   li $v0, 4                                               │
│   syscall                                                  │
│                                                            │
│   # العودة                                                │
│   lw $ra, 4($sp)                                          │
│   lw $fp, 0($sp)                                          │
│   addi $sp, $sp, 8                                        │
│   jr $ra                                                   │
│                                                            │
│ .data                                                      │
│ msg1: .asciiz "مرحباً بالعالم!"                           │
└─────────────────────────────────────────────────────────────┘
```

##### ميزات العرض:
- **تمييز الصيغة**: ألوان مختلفة لأنواع التعليمات
- **أرقام الأسطر**: تسهيل التنقل
- **نسخ الكود**: نسخ إلى الحافظة
- **حفظ الكود**: حفظ في ملف

### معالجة الأخطاء (Error Handling)

#### 1. أنواع الأخطاء (Types of Errors)

##### أخطاء معجمية (Lexical Errors):
```
خطأ في السطر 3، العمود 15: حرف غير معروف '@'
خطأ في السطر 7، العمود 8: كلمة محجوزة غير صحيحة 'اطبعا'
```

##### أخطاء نحوية (Syntax Errors):
```
خطأ في السطر 5، العمود 12: توقع '؛' بعد 'نص'
خطأ في السطر 8، العمود 3: توقع ']' بعد العبارات
```

##### أخطاء دلالية (Semantic Errors):
```
خطأ في السطر 6، العمود 15: متغير 'عمر' غير معرف
خطأ في السطر 9، العمود 20: عدم تطابق الأنواع في العملية '+'
```

#### 2. عرض الأخطاء (Displaying Errors)

##### نافذة الأخطاء:
```
┌─────────────────────────────────────────────────────────────┐
│                        الأخطاء                             │
├─────────────────────────────────────────────────────────────┤
│ السطر │ العمود │ النوع    │ الرسالة                        │
├─────────────────────────────────────────────────────────────┤
│ 3     │ 15     │ معجمي   │ حرف غير معروف '@'              │
│ 5     │ 12     │ نحوي    │ توقع '؛' بعد 'نص'              │
│ 6     │ 15     │ دلالي   │ متغير 'عمر' غير معرف           │
└─────────────────────────────────────────────────────────────┘
```

##### ميزات عرض الأخطاء:
- **تحديد موقع الخطأ**: تمييز في محرر الأكواد
- **تصنيف الأخطاء**: حسب النوع والخطورة
- **اقتراحات الإصلاح**: نصائح لحل المشكلة
- **تنقل سريع**: الانتقال لموقع الخطأ

#### 3. إصلاح الأخطاء (Fixing Errors)

##### نصائح عامة:
1. **ابدأ بالأخطاء الأولى**: قد تسبب أخطاء أخرى
2. **تحقق من الأقواس**: تأكد من تطابقها
3. **تحقق من الفواصل المنقوطة**: مطلوبة في نهاية العبارات
4. **تحقق من أسماء المتغيرات**: تأكد من تعريفها

##### أمثلة على الإصلاح:
```
// قبل الإصلاح
متغيرات:
  اسم نص؛        // خطأ: توقع ':'

// بعد الإصلاح
متغيرات:
  اسم: نص؛       // صحيح
```
